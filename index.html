Trabajando con Secuencias de ADN

En Python, las secuencias de ADN se representan como cadenas de texto (strings).

Ejemplo real: Fragmento del gen HBB (beta-globina humana)

ATGGTGCACCTGACTCCTGAGGAGA

Contar Nucleótidos

# Secuencia real de ADN
dna_seq = "ATGGTGCACCTGACTCCTGAGGAGA"

# Contar cada base
count_A = dna_seq.count('A')
count_T = dna_seq.count('T')
count_G = dna_seq.count('G')
count_C = dna_seq.count('C')

print(f"A: {count_A}, T: {count_T}, G: {count_G}, C: {count_C}")
# Resultado: A: 9, T: 4, G: 8, C: 4

Transcripción: ADN → ARN

La transcripción convierte ADN en ARN reemplazando Timina (T) por Uracilo (U).

# Transcribir ADN a ARN (T → U)
dna_seq = "ATGGTGCACCTGACTCCTGAGGAGA"
rna_seq = dna_seq.replace('T', 'U')

print("ADN:", dna_seq)
# ADN: ATGGTGCACCTGACTCCTGAGGAGA

print("ARN:", rna_seq)
# ARN: AUGGUGCACCUGACUCCUGAGGAGA

Nota: Este es un proceso biológico fundamental que Python puede simular con una simple operación de reemplazo.

Visualización con Matplotlib

Podemos visualizar la frecuencia de nucleótidos con gráficos de barras:

import matplotlib.pyplot as plt

dna_seq = "ATGGTGCACCTGACTCCTGAGGAGA"
counts = {
    'A': dna_seq.count('A'),
    'T': dna_seq.count('T'),
    'G': dna_seq.count('G'),
    'C': dna_seq.count('C')
}

bases = list(counts.keys())
frequencies = list(counts.values())

plt.bar(bases, frequencies, 
        color=['blue', 'orange', 'green', 'red'])
plt.title('Frecuencia de nucleótidos')
plt.xlabel('Nucleótidos')
plt.ylabel('Cantidad')
plt.show()

Resultado: Gráfico de barras mostrando A=9, T=4, G=8, C=4
